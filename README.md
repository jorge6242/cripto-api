## Installation

```bash
$ yarn
```

## Features

### - Typescript
### - DTO(Data Transfer Object) to type arguments and validation routes
### - TypeORM

## Running the app

```bash
# development
$ yarn start:dev
```

## Endpoints to use
### http://localhost:3000/currency/balances
### http://localhost:3000/currency/conversion?amount=100&symbol=EUR
